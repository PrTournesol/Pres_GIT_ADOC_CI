1. Introduction

L' Asciidoc est un langage de balisage (wiki) qui permet de générer facilement une page web ou un PDF. Le fichier porte l’extension .adoc et peut être edité avec n’importe quel éditeur de texte simple.

2. Logiciels d'édition

  • Asciidocfx

  • Notepad ++

  • le bloc note

  • Sublime text
